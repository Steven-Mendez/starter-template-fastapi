# Feature Template

This package is an inert scaffold for adding new feature packages under
`src/features/`. It is not mounted by `src/main.py` and should not be changed for
product behavior.

Use the full guide in [`docs/feature-template.md`](../../../docs/feature-template.md).

Reference implementation:

- [`src/features/kanban`](../kanban)

After copying this directory, replace every template placeholder before
registering the new feature.
